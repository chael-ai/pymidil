# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities by emailing **michael.armah@midil.io** with the subject line `[SECURITY] pymidil vulnerability`.

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Any suggested mitigations you are aware of

You can expect an acknowledgement within **48 hours** and a resolution timeline within **7 days** of confirmation.

We will credit reporters in release notes unless you request otherwise.
